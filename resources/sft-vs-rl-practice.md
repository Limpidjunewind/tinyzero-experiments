# 大厂实战：SFT 到什么程度开始做 RL

> 来源：小红书，2026-03-19

## 值得记的实践 heuristic

**判断"该切 RL 了"的信号：pass@k 明显高于 pass@1**
- 模型多次回答的通过率远大于一次，说明能力上限已经在那里，只是单次生成不稳定
- RL 的作用：把 pass@k 的能力"蒸馏"到 pass@1，让模型第一次就倾向于生成高概率正确的答案

**行业实践路径中的循环迭代（Reject Sampling Fine-tuning）**
- RL 之后，收集模型自己生成的、通过了 RL 筛选的高质量样本，回头再做一轮 SFT
- 形成：冷启动 SFT → RL → Reject Sampling SFT → RL → 螺旋式上升
- 这一步很多教程不提，但大厂实际在做
